- # Elemental Gem

    - **Type:** Wondrous Items
    - **Rarity:** uncommon
    - **Cost:** 960 gp
- **Property:** 



## Description 

This gem contains a mote of elemental energy. When you use an action to break the gem, an elemental is summoned as if you had cast the conjure elemental spell, and the gem's magic is lost.
